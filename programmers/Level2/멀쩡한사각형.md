### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(w, h) {
  return w * h - (w + h - gcd(w, h));
}

function gcd(w, h) {
  if (w % h == 0) {
    return h;
  }
  return gcd(h, w % h);
}
```

대각선 지나는 사각형 개수 공식을 참고하였다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for

```javascript
function solution(w, h) {
  const gcd = (a, b) => {
    return b === 0 ? a : gcd(b, a % b);
  };

  return w * h - (w + h - gcd(w, h));
}
```

화살표함수를 이용한 답안이다.
