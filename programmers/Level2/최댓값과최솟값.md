## 최댓값과 최솟값

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  let arr = s.split(' ');
  let max = arr[0],
    min = arr[0];

  arr.map((e) => {
    e = e * 1;
    if (max <= e) {
      max = e;
    }
    if (min >= e) {
      min = e;
    }
  });
  return min + ' ' + max;
}
```

배열에 담아서 for문을 통해 최댓값 최솟값을 구해줬다.  
`e = e * 1;` : 숫자형으로 바꿔주기위해 사용

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12939/solution_groups?language=javascript

```javascript
function solution(s) {
  const arr = s.split(' ');

  return Math.min(...arr) + ' ' + Math.max(...arr);
}
```

배열에서도 Math.min 함수가 가능..
