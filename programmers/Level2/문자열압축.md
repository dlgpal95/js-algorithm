### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  let answer = s.length;
  let arr = [...s];
  let result = [];

  if (s.length == 1) {
    return 1;
  }

  for (let j = 1; j < s.length; j++) {
    let num = 1;
    let text = '',
      word = '';
    for (let i = 0; i < s.length; i += j) {
      let sText = s.slice(i, i + j);
      if (text === sText) {
        num++;
      } else {
        word += num == 1 ? text : num + text;
        num = 1;
        text = sText;
      }
    }
    word += num == 1 ? text : num + text;

    result.push(word.length);
  }

  return Math.min(...result);
}
```

`if (s.length == 1) {` 만약 문자열 길이가 1이면 return 1  
첫번째 for문은 자를 문자열의 길이 (한개씩, 두개씩, 세개씩...)  
두번째 for문은 배열을 자를 문자열의 길이만큼 잘라서 앞문자랑 비교하는 것이다.

1. 앞 문자와 같으면 : num을 증가시켜준다.
2. 앞 문자와 다르면 : num+ 앞문자를 word에 붙여준다 ( 이때 1은 생략해야하기때문에 삼항연산자로 걸러준다.)

다르면 num을 초기화시키고 앞문자변수에 현재문자를 넣어준다.  
마지막 문자는 word에 이어붙일 수 없기때문에 for문을 끝나고 붙여준다.  
그러고 result배열에 길이만 넣어준 후 모든 for문이 끝나면 가장 작은 값을 리턴해준다.
