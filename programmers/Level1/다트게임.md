### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(dartResult) {
  var num = 0,
    temp = 0;
  var result = [];

  for (var i = 0; i < dartResult.length; i++) {
    if (dartResult[i] >= 0 && dartResult[i] <= 9) {
      if (dartResult[i] == 1 && dartResult[i + 1] == 0) {
        temp = 10;
        i++;
      } else {
        temp = dartResult[i];
      }
    } else if (dartResult[i] === 'S') {
      result.push(temp);
    } else if (dartResult[i] === 'D') {
      result.push(Math.pow(temp, 2));
    } else if (dartResult[i] === 'T') {
      result.push(Math.pow(temp, 3));
    } else if (dartResult[i] == '#') {
      result[result.length - 1] *= -1;
    } else if (dartResult[i] == '*') {
      result[result.length - 1] *= 2;
      result[result.length - 2] *= 2;
    }
  }

  return result.reduce((acc, cur) => (acc += parseInt(cur)), 0);
}
```

for문을 이용하여 숫자인지 문자인지 체크해준다.

숫자 일때 `if (dartResult[i] >= 0 && dartResult[i] <= 9) {` 체크해준다.
`if (dartResult[i] == 1 && dartResult[i + 1] == 0) {` 하는 이유는 10이 나올 경우를 대비해서 이다.  
이 경우에는 10값을 넣어주고 `i++`하여 다음 인덱스를 검사하지않는다.

문자 일때 각 해당 조건을 해준다.
마지막으로 배열안에 있는 값들을 더해준다.
