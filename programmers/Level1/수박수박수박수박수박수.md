### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  let answer = '';
  const watermelon = '수박';

  for (let i = 1; i <= n; i++) {
    i % 2 == 0
      ? (answer += watermelon.slice(1))
      : (answer += watermelon.slice(0, 1));
  }

  return answer;
}
```

`수박` string 을 정의해놓고 n의 숫자까지 홀수 짝수를 나눠서 append해주는 코드이다.
