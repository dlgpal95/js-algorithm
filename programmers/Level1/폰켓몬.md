### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(nums) {
  let arr = [];

  for (let val of nums) {
    if (arr.indexOf(val) == -1) {
      arr.push(val);
    }
  }

  return nums.length / 2 <= arr.length ? nums.length / 2 : arr.length;
}
```

중복체크해서 배열에넣은 다음에 num.length랑 비교해서 return해줬다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/1845/solution_groups?language=javascript

```javascript
function solution(nums) {
  const noDuplicatePokemon = new Set(nums);
  const pokemonVarietyCount = noDuplicatePokemon.size;
  const pokemonCounts = nums.length;
  return pokemonVarietyCount > pokemonCounts / 2 ? pokemonCounts / 2 : pokemonVarietyCount;
}
```

중복값을 없애기 위해 Set객체를 사용했다.

### Set 객체

자료형에 관계 없이 원시 값과 객체 참조 모두 유일한 값을 저장가능
