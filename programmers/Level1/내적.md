### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(a, b) {
  let answer = 0;

  for (let i = 0; i < a.length; i++) {
    answer += a[i] * b[i];
  }

  return answer;
}
```

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/70128/solution_groups?language=javascript

```javascript
function solution(a, b) {
  return a.reduce((acc, cur, idx) => (acc += cur * b[idx]), 0);
}
```

배열 2개로 배열내장함수 쓸것을 생각 못했는데.. reduce 함수를 이렇게 사용하면 됐었다.. 오오
