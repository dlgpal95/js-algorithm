## 정수 제곱근 판별

<hr/>

### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  const num = Math.sqrt(n);
  return num == Math.floor(num) ? Math.pow(num + 1, 2) : -1;
}
```

`Math.sqrt(n);`을 이용해서 제곱근을 구해주고 ` Math.floor(num)`을 이용해서 소수점을 버린값이랑 같으면  
존재하는 값이니까 제곱근+1의 제곱을 해주고 아니면 -1을 리턴한다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

```javascript
function nextSqaure(n) {
  var result = 0;
  var n = Math.sqrt(n);
  result = Number.isInteger(n) ? Math.pow(n + 1, 2) : 'no';
  return result;
}
```

`isInteger`함수를 이용해서 int값인지 구분을 해준 후 계산한다.
