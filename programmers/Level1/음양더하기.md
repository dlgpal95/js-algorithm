### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(absolutes, signs) {
  let answer = 0;

  absolutes.map((e, i) => {
    answer += signs[i] ? e : -e;
  });

  return answer;
}
```

반복문으로 더해준다.

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/76501/solution_groups?language=javascript

```javascript
function solution(absolutes, signs) {
  return absolutes.reduce((acc, v, i) => (acc += v * (signs[i] ? 1 : -1)), 0);
}
```

reduce함수를 사용한 답안이다.
