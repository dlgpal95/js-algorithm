### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(s) {
  let result = '';
  let arr = s.split(' ');

  for (let i = 0; i < arr.length; i++) {
    result +=
      ' ' +
      [...arr[i]]
        .map((e, i) => {
          if (e == ' ') return ' ';
          return i % 2 != 0 ? e.toLowerCase() : e.toUpperCase();
        })
        .join('');
  }

  return result.substring(1);
}
```

배열로 바꿔준다음에 for문을 돌리고 배열안에있는 글자들을 또 for문을 돌려주면서 바꿔줬다.  
단어사이에는 공백이 필요하기 때문에 `result += ' '`를 해줬는데 처음은 지워야하니까 `substring(1)`을 사용했다.  
너무어렵게 푼 느낌이다..

### :heavy_check_mark: 참고하면 좋은 다른 답안

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12930/solution_groups?language=javascript

```javascript
function toWeirdCase(s) {
  return s
    .split(' ')
    .map(i =>
      i
        .split('')
        .map((j, key) => (key % 2 === 0 ? j.toUpperCase() : j))
        .join('')
    )
    .join(' ');
}
```

내가짠 코드를 한줄로 한 코드이다..  
`join(' ')`를 이용해서 공백을 글자사이에만 넣어주었다.

### :heavy_check_mark: 참고하면 좋은 다른 답안2

Thanks for https://programmers.co.kr/learn/courses/30/lessons/12930/solution_groups?language=javascript

```javascript
function toWeirdCase(s) {
  var result = '';

  for (var word of s.split(' ')) {
    for (var i in word) {
      result += word[i][parseInt(i) % 2 == 0 ? 'toUpperCase' : 'toLowerCase']();
    }
    result += ' ';
  }

  return result.slice(0, -1);
}
```

이중 for문을 이용해서 한 코드이다.  
`result += word[i][parseInt(i) % 2 == 0 ? "toUpperCase" : "toLowerCase"]();` 이부분에서 함수를 호출하는 방법이 획기적이다..
