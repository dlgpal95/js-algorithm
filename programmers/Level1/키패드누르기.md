```javascript
function solution(numbers, hand) {
  let answer = [];

  const key = [
    ['1', '2', '3'],
    ['4', '5', '6'],
    ['7', '8', '9'],
    ['*', '0', '#']
  ];

  let left = [3, 0], // *
    right = [3, 2]; // #

  for (let item of numbers) {
    for (let i = 0; i < key.length; i++) {
      for (let j = 0; j < key[0].length; j++) {
        if (item == key[i][j]) {
          const sum = [i, j];
          // 맨 왼쪽인 경우
          if (j == 0) {
            left = sum;
            answer.push('L');
            // 맨 오른쪽인 경우
          } else if (j == 2) {
            right = sum;
            answer.push('R');
            // 가운데인 경우
          } else {
            // 왼쪽 , 오른쪽 거리가 같지않은 경우
            if (Math.abs(i - right[0]) + Math.abs(j - right[1]) > Math.abs(i - left[0]) + Math.abs(j - left[1])) {
              left = sum;
              answer.push('L');
            } else if (
              Math.abs(i - right[0]) + Math.abs(j - right[1]) <
              Math.abs(i - left[0]) + Math.abs(j - left[1])
            ) {
              right = sum;
              answer.push('R');
              // 왼쪽 , 오른쪽 거리가 같은 경우
            } else {
              // 오른손잡이
              if (hand == 'right') {
                right = sum;
                answer.push('R');
                // 왼손잡이
              } else {
                left = sum;
                answer.push('L');
              }
            }
          }
        }
      }
    }
  }

  return answer.join('');
}
```

해당 키패드를 2차원배열에 넣은 다음에 계산하였다.
numbers를 for문으로 하나씩 비교하였고 이중for문을 이용해서 해당 값을 찾았다.
값이 해당하는 인덱스를 왼쪽 or오른쪽에 넣은 후 계속 진행하였다.
for문과 if문이 너무 많아서 가독성은 떨어진다..  
다른답안보니 함수를 이용해서 사용했던데 다음에는 함수를 이용해야겠다.
