### :heavy_check_mark: 내가 푼 답안

```javascript
function solution(n) {
  return [...n].reduce((acc, curr) => parseInt(acc) + parseInt(curr), 0);
}
```

이렇게했는데 자꾸 에러가 나고 ㅠㅠ안돼서 뭐지뭐지했는데..

```javascript
function solution(n) {
  return [...(n + '')].reduce((acc, curr) => acc + parseInt(curr), 0);
}
```

... `[...n+''].` string으로 형변환을 안해줬다 ㅠㅠㅠ string으로 넘어오는줄알았는데 아니였다..ㅠㅠㅠ
그리고 acc 는 초기값 0으로 해줬으니까 parseInt할 필요가 없었다..
